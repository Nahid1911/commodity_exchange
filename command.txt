Install Rails: rails new .
Install TailwindCSS: gem "tailwindcss-rails"
                     bundle Install
                     rails tailwindcss:install
Scaffold: rails g scaffold User user_name:string user_nid:integer user_mobile:integer user_photo:string
Install Devise:
Add to Gemfile: gem 'devise'
    Run Bundle install
    rails generate devise:install
    rails g devise user
    rails db:migrate
